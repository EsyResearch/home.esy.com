# Visual Audit Reports

This directory contains visual asset audit reports generated by the `@agents/visual-auditor-agent.md` for Esy.com visual essays and scrollytelling experiences.

---

## Purpose

Visual Audit Reports serve as:
1. **Quality Records** — Documentation of SVG/animation quality for each published piece
2. **Certification** — Formal approval that visual assets meet publication standards
3. **Remediation Tracking** — Issues identified and fixes verified

## Grading System

| Grade | Score | Publication Status |
|-------|-------|-------------------|
| A+ | 97-100 | ✅ Ship immediately |
| A | 93-96 | ✅ Ship |
| A- | 90-92 | ✅ Ship with notes |
| B+ | 87-89 | ✅ Ship after review |
| B | 83-86 | ⚠️ Revise if time permits |
| B- | 80-82 | ⚠️ Revise recommended |
| C+ | 77-79 | ❌ Revise required |
| C | 73-76 | ❌ Major revision required |
| <C | <73 | ❌ Reject |

## Generating New Reports

To audit visual assets for a visual essay:

```
Using @agents/visual-auditor-agent.md, audit the SVG illustrations for
the visual essay at /essays/visual/{slug}
```

The agent will produce:
1. Five-tier comprehensive audit
2. Individual SVG grades
3. Blocking issues (if any)
4. Recommendations
5. Certification status
