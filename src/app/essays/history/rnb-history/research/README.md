## Source Tier Classification

| Tier | Description | Examples |
|------|-------------|----------|
| **Tier 1** | Academic/Primary | Peer-reviewed journals, university presses, archives, autobiographies |
| **Tier 2** | Highly Credible | Major news (original reporting), documentaries, encyclopedias |
| **Tier 3** | Supplementary | Wikipedia (verify elsewhere), expert blogs |
| **Tier 4** | Avoid | Social media, anonymous sources, content farms |
