## Executive Summary

With a $0 budget, this report identifies all freely available imagery for "Hip-Hop: From the Bronx to the World."

**Reality Check**: Hip-Hop is a 50-year-old culture. Most iconic photography is copyrighted by living photographers. True public domain Hip-Hop imagery is extremely limited.

**Strategy**: Combine the few available CC-licensed Hip-Hop photos with:
- Contemporary urban/graffiti photography (Unsplash/Pexels)
- Older historical Bronx imagery (NYPL, pre-1928)
- Location photography (1520 Sedgwick today, etc.)
- Creative visual alternatives (maps, typography, illustrated elements)

## Alternative Visual Approaches

### 1. Typography-Driven Design
Use the strong typography system (Impact/Anton) to carry chapters where photography is unavailable. Large quotes, lyric excerpts, and typographic compositions.

### 2. Custom Illustrations/SVGs
Commission or create:
- Turntable illustrations
- Bronx map graphics
- Timeline visualizations
- Element iconography (DJ, MC, Breaking, Graffiti, Knowledge)

### 3. Contemporary Location Photography
Take or source new photos of:
- 1520 Sedgwick Avenue (today)
- Cedar Park
- South Bronx streets
- NYC subway system

### 4. Silhouette/Outline Approach
Use silhouetted figures rather than specific portraits—evocative without requiring licensed photos.

### 5. Archive Texture + Typography
Layer historical textures (brick, concrete, subway tile) with typographic content.

## Honest Assessment

### Strengths of $0 Approach:
- Forces creative visual solutions
- Contemporary imagery maintains fresh, living-culture feel
- Typography-driven design can be powerful
- No licensing complications

### Limitations:
- Cannot show most iconic Hip-Hop figures in historical context
- Essay loses "definitive visual history" character
- Missing the raw 1970s-80s photography that defines the era
- May feel less authoritative without archival imagery

### Recommendation:

If the goal is a **definitive visual history**, some budget for essential images (DJ Kool Herc early party, one subway graffiti piece, one breaking cipher) would strengthen the work significantly. Even $500-1,000 could secure a few critical shots.

If the goal is a **contemporary reflection on Hip-Hop's legacy**, the $0 approach with modern photography and creative typography can work—but the essay should be reframed accordingly.

## Appendix: Direct URL Extraction

### Wikimedia Commons
```bash
# Get direct image URL from file page:
curl -s "https://commons.wikimedia.org/wiki/File:Example.jpg" | \
  grep -oE 'https://upload\.wikimedia\.org/wikipedia/commons/[a-f0-9]/[a-f0-9]{2}/[^"]+\.(jpg|png)' | \
  grep -v thumb | head -1
```

### Unsplash
Direct download links available on each image page. No extraction needed.

### Verification
```bash
curl -sI "IMAGE_URL" | grep -i content-type
# Should return: content-type: image/jpeg
```
