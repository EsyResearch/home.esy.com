# Era Visual Treatment Guide: Hip-Hop History

## Purpose

This document defines distinct visual treatments for each historical era, ensuring the essay feels like a time-traveling documentary. Each era has:
- Color palette
- Typography direction
- Texture and grain
- Image processing rules
- UI/layout adjustments
- Cultural reference points

---

## Era 1: Pre-Hip-Hop DNA (1940s-1972)

### Time Period
Sound system culture in Jamaica, funk/soul in America, spoken word traditions

### Color Palette

| Color | Hex | Usage |
|-------|-----|-------|
| Caribbean Blue-Green | #00CED1 | Sound system accent |
| Jamaica Yellow | #FED100 | Highlight, warmth |
| Kingston Night | #1C1C1C | Backgrounds |
| Tropical Orange | #FF6B35 | Secondary accent |
| Sepia Base | #D4A574 | Text overlays |

### Typography Direction
- **Headlines**: Afro-futurist display fonts (Klim Heldane Display or similar)
- **Body**: Classic serif (Garamond, Minion)
- **Accents**: Hand-lettered Caribbean signage aesthetic
- **Feel**: Warm, organic, pre-digital

### Texture & Grain
- Warm sepia overlays on photographs
- Vinyl record textures (grooves, labels)
- Sound system speaker cloth patterns
- 1960s magazine halftone dots

### Image Processing
- **Archival**: Preserve original warmth; enhance without modernizing
- **Color**: Kodachrome-era color casts (warm yellows, orange shadows)
- **B&W**: High-contrast with warm toning
- **Grain**: Fine grain, period-appropriate

### UI Adjustments
- Rounded corners suggesting radio dials
- Sound wave visualizations as dividers
- Record label circular elements

### Cultural References
- Jamaican sound system advertising
- 1960s funk concert posters
- Last Poets album covers
- Radio DJ booth aesthetics

### Time Period
First parties, DJ development, MC emergence, B-boy beginnings

### Color Palette

| Color | Hex | Usage |
|-------|-----|-------|
| Bronx Brick | #8B4513 | Warm accent |
| Urban Decay | #4A4A4A | Backgrounds |

| 70s Cream | #FFF8DC | Text, light elements |
| Ghetto Gold | #DAA520 | Accents |

### Typography Direction
- **Headlines**: Condensed industrial sans (Helvetica Compressed, Impact)
- **Body**: Clean sans-serif (Helvetica, Univers)
- **Accents**: Hand-drawn, Sharpie-marker style
- **Feel**: Raw, street-level, handmade

### Texture & Grain
- Photocopy/xerox grain (party flyers)
- Brick wall textures
- Asphalt and concrete
- Gym floor wood grain
- Burned building rubble

### Image Processing
- **Archival**: High contrast, gritty
- **Color**: 1970s film stock (Ektachrome, muted)
- **B&W**: Documentary style, harsh flash
- **Grain**: Prominent grain, period cameras

### UI Adjustments
- Flyer-style torn edges
- Tape/staple marks on images
- Handwritten captions
- Gym rec room signage aesthetic

### Cultural References
- Block party flyers (Phase 2 designs)
- 1970s South Bronx photography (Joe Conzo)
- Rec room and community center signage
- NYC subway map colors

- **Headlines**: Art Deco/Disco display (Broadway, Pump Triline)

- VU meter graphics

- Vinyl sleeve corners

### Cultural References

- Studio photography

## Transition Treatments

### Era-to-Era Transitions

When moving between eras, use:

1. **Color Fade**: Gradual palette shift over 2-3 scroll sections
2. **Typography Evolution**: Headline font morphs between eras
3. **Grain Transition**: Texture grain adjusts smoothly
4. **Timeline Marker**: Visual timeline progress bar updates

### Cross-Era Consistency

Maintain across all eras:
- Source citation style
- Caption lockup format
- Progress bar (turntable/waveform motif)
- Navigation elements
- Accessibility standards

## Implementation Notes

### CSS Custom Properties

```css
/* Era-based theming */
:root {
  /* Era 2: Bronx Genesis (default starting era) */
  --era-primary: #FF8C00;
  --era-secondary: #4A4A4A;
  --era-accent: #DAA520;
  --era-text: #FFF8DC;
  --era-grain: 0.4;
  --era-grain-type: 'xerox';
}

[data-era="golden-age"] {
  --era-primary: #D21034;
  --era-secondary: #000000;
  --era-accent: #FFD700;
  --era-text: #FFFFFF;
  --era-grain: 0.3;
  --era-grain-type: 'film';
}
```

### Performance Considerations

- Preload era assets during previous section scroll
- Use CSS for texture overlays where possible (not image overlays)
- Lazy-load era-specific fonts
- Transition durations: 300-500ms for era shifts
