## URL Pattern Reference

### LOC Gottlieb Collection (IIIF)
```
https://tile.loc.gov/image-services/iiif/public:music:musgottlieb-{ID}:0001/full/pct:{PERCENT}/0/default.jpg
```
- `pct:100.0` = Full resolution
- `pct:50.0` = Half resolution
- `pct:25.0` = Quarter resolution

### LOC Van Vechten Collection
```
https://tile.loc.gov/storage-services/service/pnp/van/{folder}/{subfolder}/{id}r.jpg
```
- `r.jpg` suffix = Full resolution
