## IMAGE SPECIFICATIONS

### Required Formats
```
Hero images: 1920x1080 minimum, WebP + JPEG fallback
Inline images: 800px width standard, 1600px retina
Thumbnails: 400x400 cropped
Logos: SVG preferred, PNG fallback
```

### Alt Text Requirements
- All images require descriptive alt text
- Korean names include romanization
- Historical context included for archival images

### Attribution Format
```
[Image description]. Source: [Platform]. License: [Type].
Credit: [Photographer/Creator if known].
```

## LICENSING COMPLIANCE CHECKLIST

### Before Using Any Image:
- [ ] Verify license type on source page
- [ ] Confirm commercial/editorial use allowed
- [ ] Note attribution requirements
- [ ] Check for personality rights restrictions
- [ ] Document source URL and access date

### South Korean Personality Rights Note
South Korean law provides strong personality rights protection. Images of identifiable individuals may require:
- Editorial context (news reporting, criticism, commentary)
- No commercial endorsement implication
- Respectful presentation
- Proper attribution

This essay qualifies as educational/critical commentary under fair use principles.
