# Key Figures: The Word That Changed Its Stripes

> Research for Visual Essay on Etymology of "Dick"

## Figure Profiles

### Figure 1: Richard I (Richard the Lionheart)
| Field | Value |
|-------|-------|
| **Name** | Richard I of England |
| **Epithet** | The Lionheart / Cœur de Lion |
| **Dates** | 1157–1199 |
| **Role** | King who elevated the name Richard in England |
| **Contributions** | His legendary status made "Richard" one of the most prestigious names in medieval England; set the stage for nicknames like Dick |
| **Quote** | "I would sell London if I could find a buyer." (Attributed, raising Crusade funds) |
| **Photography** | Medieval manuscript illuminations, stone effigies at Fontevraud Abbey |
| **Era Treatment** | Medieval warm gold, aged parchment aesthetic |

### Figure 2: Francis Grose
| Field | Value |
|-------|-------|
| **Name** | Francis Grose |
| **Epithet** | The Vulgar Lexicographer |
| **Dates** | 1731–1791 |
| **Role** | English antiquary and lexicographer |
| **Contributions** | Published "A Classical Dictionary of the Vulgar Tongue" (1785), first serious documentation of English street slang; revealed hidden vocabulary of Georgian England |
| **Quote** | "The Vulgar Tongue... words and phrases used by... persons of low and illiberal education." (Dictionary preface) |
| **Photography** | 18th-century portrait engravings available |
| **Era Treatment** | Georgian sage/muted tones, period typography |

## Figure Selection Rationale

| Figure | Era Represented | Why Essential |
|--------|-----------------|---------------|
| Richard I | Medieval (origin) | Name's prestige source |
| Francis Grose | Georgian (documentation) | First slang dictionary |
| John S. Farmer | Victorian (attestation) | First anatomical documentation |
| Eric Partridge | 20th c. (scholarship) | Authoritative etymology |
| Richard Nixon | Modern (decline) | Last famous "Dick" |
| Dick Van Dyke | Contemporary (survival) | Proves context matters |
| Norman Bogner | 1960s (shift) | First pejorative attestation |
