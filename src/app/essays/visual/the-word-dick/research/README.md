## Key Findings

1. **The rhyming nickname pattern (Richard → Rick → Dick) dates to the 13th century** — established medieval practice
2. **Generic "fellow" meaning by 1550s** — name's popularity made it generic
3. **First anatomical attestation: 1891** — Farmer's Slang Dictionary
4. **First pejorative attestation: 1966** — Norman Bogner's *Seventh Avenue*
5. **Name fell out of top 1000 by 1969** — SSA data documents decline
6. **<5 babies named Dick by 2014** — effectively extinct as baby name
