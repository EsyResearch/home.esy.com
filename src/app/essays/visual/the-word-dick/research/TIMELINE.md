# Timeline: The Evolution of "Dick"

> Research for Visual Essay on Etymology of "Dick"

## Chronological Events

### Era 1: Medieval Origins (1066–1500)

| Year | Event | Significance | Source |
|------|-------|--------------|--------|
| **1066** | Norman Conquest | Richard (from Germanic "Ricohard") arrives in England | Historical |
| **c. 1200s** | Rhyming nickname pattern established | Richard → Rick → Dick pattern emerges (parallels William → Will → Bill) | Etymonline |
| **1157–1199** | Richard I reigns | Richard the Lionheart elevates the name's prestige | Historical |
| **1300s** | Dick common nickname | Fully established as standard pet name for Richard | Etymonline |

### Era 2: Tudor & Stuart — The Generic Man (1500–1700)

| Year | Event | Significance | Source |
|------|-------|--------------|--------|
| **c. 1550s** | Dick = generic "fellow" | Name becomes so common it means "any man" | Etymonline |
| **1553** | First recorded "generic" usage | Earliest documented use of Dick meaning "man/fellow" | Etymonline |
| **c. 1600s** | "Tom, Dick, and Harry" emerges | Phrase becomes standard for "any ordinary people" | Multiple sources |

### Era 3: Georgian — The Documentation (1700–1840)

| Year | Event | Significance | Source |
|------|-------|--------------|--------|
| **1785** | Grose publishes "Classical Dictionary of the Vulgar Tongue" | First serious English slang dictionary | Historical |
| **c. 1790s** | Anatomical meaning in circulation | Vulgar meaning exists in oral usage, barracks slang | Indirect evidence |

### Era 4: Victorian — The Attestation (1840–1900)

| Year | Event | Significance | Source |
|------|-------|--------------|--------|
| **1891** | Farmer's "Slang and Its Analogues" | **First documented anatomical meaning in print** | Farmer via Etymonline, OED |
| **1890s** | Meaning bifurcates | Name remains respectable; slang circulates in lower registers | Historical context |

### Era 5: Modern — The Expansion (1900–1970)

| Year | Event | Significance | Source |
|------|-------|--------------|--------|
| **1908** | "Dick" = detective first recorded | American slang for detective/investigator emerges | Etymonline |
| **1930s–1940s** | Peak name popularity | "Dick" reaches maximum popularity as given name in USA | SSA data |
| **1966** | First pejorative attestation | Norman Bogner's *Seventh Avenue*: "He's a dick" | Grammarphobia |
| **1969** | Falls out of top 1000 names | Name begins rapid decline in USA | SSA via Wikipedia |

### Era 6: Contemporary — The Taboo (1970–Present)

| Year | Event | Significance | Source |
|------|-------|--------------|--------|
| **1970s** | Vulgar meaning becomes dominant | Anatomical/pejorative meanings eclipse the name | Cultural shift |
| **1974** | Nixon resigns | One of last famous "Dicks" leaves public stage | Historical |
| **1980s** | Name nearly extinct for newborns | Parents actively avoid the name | SSA trends |
| **2014** | <5 babies named Dick | Virtually no American babies receive the name | SSA via Wikipedia |
| **Present** | Word fully bifurcated | "Dick" as name survives only in older generations; slang dominates | Current usage |

---

## Key Turning Points

| Turning Point | Date | What Changed |
|---------------|------|--------------|
| **Rhyming Pattern Established** | c. 1200s | Created the nickname that would eventually transform |
| **Generic Meaning Emerges** | c. 1550s | Name became so common it meant "any man" |
| **First Print Attestation (Vulgar)** | 1891 | Meaning crossed from oral to documented |
| **Detective Slang Emerges** | 1908 | Third major meaning branch appears |
| **Pejorative Crystallizes** | 1966 | Character insult meaning documented |
| **Name Popularity Collapse** | 1969 | Crossed threshold into unacceptability |
