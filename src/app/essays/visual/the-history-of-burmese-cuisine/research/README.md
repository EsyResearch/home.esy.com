# Research Directory

This directory contains research documentation for **The History of Burmese Cuisine** visual essay.

## Contents

| File | Purpose |
|------|---------|
| `CITATIONS.md` | **Citation log** — Single source of truth for all citations |

## Purpose

The research directory ensures:
1. **Traceability** — Every claim in the essay can be traced to a source
2. **Verifiability** — All sources are logged with verification status
3. **Auditability** — Citation audits can reference this canonical source list
4. **Maintainability** — Broken links or outdated sources can be identified and fixed

## Citation Log Structure

The `CITATIONS.md` file contains:

- **Sources Section Citations** — All sources displayed in the essay
- **Image Citations** — Summary of image licensing (details in `images.ts`)
- **Quotes & Attributions** — Status of all quoted/attributed statements
- **Factual Claims** — Major claims requiring citation support
- **Version History** — Changes to citations over time
- **Audit Trail** — Record of citation audits
