# Visual Sources & Typographic References: "SLANG" — Typographic Visual Essay

## Overview

Since v2 prohibits photographs, this document catalogs **typographic references** and **abstract visual sources** that can inspire or guide the designer. All historical "artifacts" will be recreated typographically.

---

## Typography Reference Sources

### Georgian Era (1750–1820)

#### Reference 1: Caslon Typefaces
- **Description:** William Caslon's types dominated English printing in the 18th century
- **Relevant For:** 1756 attestation panel, Grose dictionary recreation
- **Characteristics:** Moderate contrast, warm bracketed serifs, classical proportions
- **Modern Equivalents:** Adobe Caslon Pro, ITC Founders Caslon
- **License Status:** Commercial fonts available

#### Reference 2: Long S (ſ)
- **Description:** Historical letterform used in printed English until ~1800
- **Relevant For:** Period-accurate facsimiles
- **Usage:** Internal "s" was often ſ, final "s" was modern s
- **Example:** "claſſical" for "classical"

#### Reference 3: Grose's 1785 Dictionary Typography
- **Description:** Typical late 18th-century dictionary layout
- **Source:** Public domain scans available via Internet Archive
- **Characteristics:** Two-column, small caps for headwords, tight leading
- **Recreation Guidance:** Use period-appropriate serifs, replicate entry structure

---

### Victorian Era (1850–1910)

#### Reference 4: Victorian Display Types
- **Description:** Bold, decorative types for headlines and posters
- **Relevant For:** Hotten's 1859 dictionary title, Victorian-era panels
- **Characteristics:** Fat faces, Tuscan styles, elaborate ornamentation
- **Modern Equivalents:** Playfair Display, Bodoni-style fonts
- **License Status:** Many free alternatives

#### Reference 5: Encyclopædia Britannica Typography (11th ed.)
- **Description:** The authoritative reference style of 1911
- **Relevant For:** Bradley's definition panel
- **Characteristics:** Formal serif, justified columns, hierarchical structure
- **Recreation Guidance:** Rigid grid, minimal ornament, scholarly austerity

---

### American Era (1919–1960)

#### Reference 6: American Newspaper Typography
- **Description:** Bold, condensed headlines of early 20th-century press
- **Relevant For:** Mencken panels, American champions section
- **Characteristics:** High contrast, tight tracking, exclamatory energy
- **Modern Equivalents:** Franklin Gothic, Knockout, Oswald
- **License Status:** Various free/commercial options

#### Reference 7: Book Typography of the Period
- **Description:** Clean, readable text settings of quality publishers
- **Relevant For:** Body text in scholarly sections
- **Characteristics:** Generous margins, clear hierarchy, classical proportions
- **Modern Equivalents:** Crimson Text, EB Garamond, Libre Baskerville

---

### Contemporary Era (1996–2019)

#### Reference 8: Academic Press Typography
- **Description:** Oxford University Press and similar scholarly styles
- **Relevant For:** Coleman, Eble, Green panels
- **Characteristics:** Professional, restrained, clearly hierarchical
- **Modern Equivalents:** Source Serif Pro, Merriweather

#### Reference 9: Digital Interface Typography
- **Description:** Web-native type systems
- **Relevant For:** Internet era panels, McCulloch section
- **Characteristics:** Clean sans-serifs, responsive, accessible
- **Modern Equivalents:** Inter, Space Grotesk, IBM Plex Sans
- **Note:** Vary from typical choices; consider Söhne, Satoshi, General Sans

## Abstract Visual References

### Reference 10: Typographic Posters (Swiss/International Style)
- **Description:** Mid-20th-century poster design emphasizing type as image
- **Relevant For:** Overall visual language, panel compositions
- **Key Features:** Grid-based, type at scale, limited color, negative space
- **Artists to Study:** Josef Müller-Brockmann, Armin Hofmann

### Reference 11: Contemporary Editorial Design
- **Description:** Magazine and book designs that prioritize typography
- **Relevant For:** Modern panel aesthetics

### Reference 12: Museum Exhibition Graphics

## Graphic Motif Sources

### Reference 13: Brackets and Editorial Marks
- **Description:** Typographic conventions for interpolation, editorial comment
- **Relevant For:** [SLANG] labeling, scholarly annotation motif
- **Design Usage:** Square brackets for contested terms, angle brackets for reconstructed text

### Reference 14: Redaction Aesthetics
- **Description:** Blacked-out text as visual device
- **Relevant For:** Taboo/suppression themes, etymology uncertainty
- **Design Usage:** Black rectangles over letterforms, partial reveals

### Reference 15: Rubber Stamps
- **Description:** Authority/verification visual language
- **Relevant For:** "ATTESTED 1756" seals, date stamps
- **Design Usage:** Rounded rectangle frames, slightly irregular edges (vector)

### Reference 16: Dictionary Entry Structure
- **Description:** Formal lexicographic layout conventions
- **Relevant For:** Facsimile recreations, micro-artifact elements
