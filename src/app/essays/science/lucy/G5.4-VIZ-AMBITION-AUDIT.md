## Technology Tier

Per the G5.4 contract, the Lucy essay targets a **minimum average tier of 2.0** (VIZ-TECH-DECISIONS.md specifies an average tier of 2.83). Ten visualizations were assigned Tier 3; two were assigned Tier 2. The implementation uses a mix of Recharts, D3, and React state to achieve interactive data presentation. Several Tier 3 assignments were implemented at Tier 2 (table-based) where geographic or canvas-based solutions were deferred, but the data remains accessible and well-structured.

## Recommendations for Future Iteration

1. **D3 (Hadar Site Map):** Implement D3-Geo projection with simplified Africa TopoJSON. Add click-to-reveal locality panels. Consider static map image as intermediate step if full D3-Geo is costly.

2. **D6 (Contemporary Species Map):** Add D3-Geo base map with time-slider or auto-play for animated species ranges. The presence/absence table could remain as a companion view for accessibility.

3. **D10 (Bipedalism Biomechanics):** Implement Canvas-based gait animation with requestAnimationFrame. Side-by-side comparison of chimp, Lucy, and human gait cycles would meet the Tier 3 assignment. Ensure cleanup on unmount to prevent memory leaks.

4. **D1 (Anatomical Comparison):** Consider D3/SVG overlay with side-by-side silhouettes or measurement annotations. Hover tooltips on trait rows could enhance the current table.

5. **D2 (Climate Timeline):** Add D3 brush for zoomable time range if Recharts does not support it natively. Multi-layered series (CO₂, species ranges) could be added per original spec.

6. **D4 (Dietary Analysis):** Add convex hull overlays per species cluster using d3-delaunay or manual hull computation. Would strengthen the isotope-space grouping narrative.

7. **D9 (Cause of Death):** Add clickable fracture sites on a skeleton SVG diagram, linking to the table rows. Would align with the "clickable fracture sites" spec.

8. **Code splitting:** Per VIZ-TECH-DECISIONS.md, consider lazy-loading heavy D3 and Canvas components (D3, D6, D10) below the fold to improve initial load performance.
