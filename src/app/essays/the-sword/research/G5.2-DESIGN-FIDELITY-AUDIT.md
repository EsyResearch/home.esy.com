## üü° Major Deviations (Intentional Departures)

### Deviation 1: Scroll-Lock Sequences Not Implemented
- **Category**: Interaction / Animation
- **Spec Requirement**: Invocation spec defines 7 scroll-lock sequences (hero forge birth, bronze casting, pattern-welding, three-path comparison, blade collision, forging meditation, museum glass transformation)
- **Implementation**: Static sections with intersection-observer reveals; no viewport locking or scroll-driven animation
- **Location**: All sections use standard scroll flow
- **Impact**: Reduces interactivity ambition but improves accessibility and performance
- **Classification**: **Intentional Departure** ‚Äî The implementation prioritizes readability and broad device compatibility over complex scroll-lock interactions. The narrative remains intact and the essay is more accessible to users on low-powered devices or with motion sensitivity.
- **Rationale Assessment**: **SOUND** ‚Äî Scroll-lock sequences are technically complex, performance-intensive, and can be disorienting for some users. The implemented approach maintains the essay's narrative structure while ensuring broader accessibility. The forging sequence is still present as a stepped component with visual progression.
- **Recommendation**: **ACCEPT** ‚Äî This departure improves the essay without compromising its core identity. The metallurgical aesthetic and narrative arc remain fully intact.

### Deviation 2: Parallax Effects Absent
- **Category**: Interaction / Animation
- **Spec Requirement**: DESIGN-RESEARCH.md specifies parallax with 0.05-0.15 factor for subtle depth; invocation spec defines 5-layer parallax system (0.2x background, 0.5x mid-ground, 1.0x subject, 1.3x overlay, 1.6x ambient)
- **Implementation**: No parallax effects implemented; all elements scroll at 1.0x
- **Location**: All sections
- **Impact**: Reduces visual depth but improves performance and readability
- **Classification**: **Intentional Departure** ‚Äî Parallax can cause motion sickness, reduce text legibility, and impact performance on mobile devices. The implementation prioritizes clarity and accessibility.
- **Rationale Assessment**: **SOUND** ‚Äî DESIGN-RESEARCH.md itself notes "Swords are heavy objects; excessive parallax would make them feel weightless" and recommends subtle 0.05-0.15 factor. The implementation takes this principle to its logical conclusion: zero parallax maintains maximum visual weight.
- **Recommendation**: **ACCEPT** ‚Äî The absence of parallax does not compromise the essay's identity. The dark backgrounds, weighty typography, and deliberate animations maintain the "forged" aesthetic without requiring parallax depth.

### Deviation 3: Hamon Line Draw Animation Missing
- **Category**: Interaction / Animation
- **Spec Requirement**: DESIGN-RESEARCH.md specifies "Hamon line draw" animation at 1200ms with `cubic-bezier(0.65, 0, 0.35, 1)` as "the essay's signature animation"
- **Implementation**: Hamon divider is static gradient line; no draw animation
- **Location**: `.sword-hamon-divider__line` in CSS
- **Impact**: Reduces visual drama but maintains the hamon metaphor
- **Classification**: **Intentional Departure** ‚Äî The hamon divider still functions as a section separator and visual metaphor for the temper line. The static implementation is simpler and more performant.
- **Rationale Assessment**: **ACCEPTABLE** ‚Äî While the spec identifies this as a signature animation, the static hamon divider still serves its purpose as a visual motif derived from sword-making. The draw animation would be a refinement rather than a requirement.
- **Recommendation**: **CONDITIONAL ACCEPT** ‚Äî The current implementation is functional. If time permits, adding the draw animation on first appearance would enhance the visual identity, but its absence does not constitute a failure.
