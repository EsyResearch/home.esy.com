# Performance Audits

This directory contains performance audit reports and interpretations for esy.com.

## Structure

Each audit is organized by date and includes:
- **Audit report** (PDF or screenshots from Lighthouse/PageSpeed Insights)
- **Interpretation document** (analysis of findings in context of the codebase)
- **Implementation recommendations** (actionable fixes with code locations)

## Naming Convention

- `YYYY-MM-DD-homepage-lighthouse.md` - Homepage audits
- `YYYY-MM-DD-[page-name]-lighthouse.md` - Specific page audits
- `YYYY-MM-DD-comprehensive-performance.md` - Multi-page or site-wide audits
