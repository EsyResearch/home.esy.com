# Orchestration Audits

This directory contains all audit reports generated by the Esy.com audit agent system.

## Directory Structure

```
orchestration/audits/
├── README.md                          # This file
├── the-thinking-machine/              # Audits for "The Thinking Machine" visual essay
│   ├── 2024-12-09-scroll-audit.md     # Scroll functionality audit
│   └── 2024-12-10-experience-audit.md # Full experience audit
├── [other-essay-slug]/                # Audits for other essays
│   └── ...
```

## Audit Types

| Type | Agent | Focus |
|------|-------|-------|
| **Scroll Audit** | `immersive-scrolling-auditor.md` | Scroll-lock, performance, mobile |
| **Experience Audit** | `immersive-experience-auditor.md` | Animations, reveals, interactions |
| **Visual Audit** | `visual-auditor-agent.md` | SVG quality, accessibility |

## Naming Convention

```
YYYY-MM-DD-[audit-type]-audit.md
```

**Examples:**
- `2024-12-09-scroll-audit.md`
- `2024-12-10-experience-audit.md`
- `2024-12-11-visual-audit.md`
- `2024-12-12-comprehensive-audit.md`

## Audit Lifecycle

1. **Triggered** — Audit requested by Visual Essay Orchestrator or user
2. **In Progress** — Audit agent(s) conducting assessment
3. **Complete** — Report generated with certification status
4. **Addressed** — Blocking issues fixed (if any)
5. **Certified** — Final approval for publication

## Certification Statuses

| Status | Meaning | Next Step |
|--------|---------|-----------|
| ✅ **CERTIFIED** | Ready for publication | Proceed to publish |
| ⚠️ **CONDITIONAL** | Minor issues to address | Fix and re-audit |
| ❌ **REJECTED** | Blocking issues present | Major fixes required |
