# Design Research Report: Palestine / Israel

> Visual Identity Development for Multi-Era Historical Scrollytelling
> Created: December 23, 2024
> Status: ✅ Ready for Production

---

## Executive Summary

This visual essay about the history of Palestine/Israel demands a **documentary archaeological aesthetic** where **the land and its layers** are the visual protagonist. The subject matter spans 4,000 years across multiple civilizations, empires, and peoples—each leaving distinct visual traces in the archaeological and documentary record.

**Design Philosophy**: The land itself is the constant; peoples come and go. The visual identity must emerge from the material culture of the region—Jerusalem stone, olive wood, ancient pottery, calligraphic traditions (Hebrew, Arabic, Greek), religious artifacts, and the documentary photography of conflict. No single cultural tradition should dominate; all must be honored.

**Critical Constraint**: This essay documents contested history. Visual choices must not privilege any side. The aesthetic must feel scholarly, archival, and documentary—not propagandistic for any perspective.

## Visual Identity Analysis

### Subject Matter Characteristics

**Subject IS**:
- Multi-civilizational (Canaanite, Israelite, Roman, Islamic, Ottoman, modern)
- Contested (competing narratives, claims, interpretations)
- Archaeological (layers upon layers of history)
- Documentary (especially for modern period)
- Sacred to multiple traditions (Jewish, Christian, Muslim)
- Ongoing (active conflict, not historical only)

**Subject Is NOT**:
- Single-tradition (must honor all)
- Resolved or conclusive (open questions remain)
- Entertainment or spectacle (demands solemnity)
- Ancient only (extends to present day)
- Propagandistic (no side should be visually privileged)

### Visual Treatment Decision

**Primary Medium**: Photorealistic (archival photography and archaeological documentation)

**Rationale**:
1. Subject demands documentary credibility—illustrations would feel invented
2. Rich archival photography exists across all eras (especially 1850s onward)
3. Archaeological photography grounds ancient claims in physical evidence
4. Multiple photographic traditions (Jewish, Arab, British, international) prevent single-perspective dominance
5. Documentary aesthetic supports scholarly tone


### Color Psychology Justification

- **Jerusalem Cream (#F5F0E6)**: The limestone that defines the city—every building, wall, and path shares this color. Warm without being political.
- **Dome Gold (#D4AF37)**: The most recognizable visual element of Jerusalem's skyline. Sacred to Muslims, visible to all.
- **Mosque Tile (#1E4D5C)**: Islamic artistic tradition without being exclusively so—also Mediterranean, ancient.
- **Olive Wood (#8B5A2B)**: Symbol of peace (olive branch) and rootedness—claimed by both peoples.
- **Terra Cotta (#C65D3B)**: Archaeological neutrality—pottery transcends political claims.
- **Conflict Red (#8B0000)**: Used only for content warnings and explicit violence markers. Never decorative.

## Typography Selection

### Philosophy

Typography must honor multiple scripting traditions without privileging any. The essay discusses Hebrew, Arabic, Greek, Latin, and Ottoman Turkish texts. The design must accommodate multilingual content while maintaining readability and scholarly credibility.

### Typography Stack

| Role | Font | Reasoning |
|------|------|-----------|
| **Display** | Playfair Display | Classical authority; recalls scholarly/documentary tradition; works across languages |
| **Body Serif** | Source Serif Pro | Highly readable; excellent multilingual support; neutral scholarly tone |
| **Body Sans** | Source Sans Pro | Clean captions, data, UI elements |
| **Hebrew** | System Hebrew (David, Narkisim) | Authentic for Hebrew quotes and terms |
| **Arabic** | System Arabic (Geeza Pro, Traditional Arabic) | Authentic for Arabic quotes and terms |
| **Monospace** | IBM Plex Mono | Dates, data, archaeological notations |

### Typography Justification

- **Playfair Display**: Evokes scholarly publishing, museum exhibitions, documentary titles. Classical without being tied to any specific tradition.
- **Source Serif Pro**: Google's highly legible serif optimized for screen reading. Neutral—doesn't favor any cultural tradition.
- **System Scripts**: Using system fonts for Hebrew and Arabic ensures authentic rendering and appropriate cultural weight.

### Typography Scale

```css
:root {
  /* Display (Chapter Titles) */
  --type-display: 2.5rem / 1.1;      /* 40px mobile */

  /* Headlines */
  --type-h1: 2rem / 1.15;            /* 32px */
  --type-h2: 1.5rem / 1.2;           /* 24px */
  --type-h3: 1.25rem / 1.3;          /* 20px */

  /* Body */
  --type-body: 1.125rem / 1.7;       /* 18px */
  --type-body-large: 1.25rem / 1.6;  /* 20px */

  /* Quotes */
  --type-quote: 1.375rem / 1.5;      /* 22px */

  /* Captions & UI */
  --type-caption: 0.875rem / 1.5;    /* 14px */
  --type-micro: 0.75rem / 1.4;       /* 12px */

  /* Timeline Dates */
  --type-date: 0.875rem / 1.4;       /* 14px, monospace */
}

@media (min-width: 768px) {
  :root {
    --type-display: 3.5rem / 1.1;
    --type-h1: 2.5rem / 1.15;
    --type-body-large: 1.375rem / 1.6;
  }
}
```

### Special Typography Treatments

**1. Multilingual Quotes**
Historical quotes displayed with original script (Hebrew/Arabic) alongside English translation. Original script slightly larger, translation below.

**2. Era Headers**
Each chapter header uses typography weight and spacing that reflects its era:
- Antiquity: Heavier, more monumental
- Classical: Formal, centered
- Modern: Documentary, left-aligned

**3. Scripture and Document Excerpts**
Quoted religious or official texts displayed in distinct style—blockquote treatment with era-appropriate background tint.

## Animation Philosophy

### Core Principle

Animation matches the nature of archaeology and documentary: **deliberate, measured, revealing**. Motion suggests excavation, document unfolding, and the patient work of historical understanding. Nothing fast, flashy, or entertaining.

### Subject Nature → Animation Approach

**Subject Is**:
- Archaeological (layers revealing layers)
- Documentary (photographs fading in, documents materializing)
- Contemplative (history demands reflection)
- Heavy (violence, displacement, grief)

**Animation Should**:
- Feel like **excavation**—slow reveals, layers uncovering
- Support **documentary credibility**—photographs appear as if developing
- Respect **emotional weight**—no bouncy or playful motion
- Enable **contemplation**—generous timing, no rushing

### Animation Principles

**Scroll-Lock Sequences**:
- Purpose: Major historical moments, map transformations, parallel narratives
- Feel: Archaeological uncovering, documentary pacing
- Timing: 600-1000px scroll depth, never rushed
- Easing: `cubic-bezier(0.4, 0, 0.2, 1)` — smooth, never bouncy

**Parallax**:
- Intensity: Subtle (this is history, not spectacle)
- Rationale: Suggests depth of time, layers of civilization
- Layers: Background 0.3x, Mid 0.6x, Subject 1x, Overlay 1.2x

**Photograph Reveals**:
- Duration: 500-700ms
- Style: Fade with subtle scale (1.02 → 1.0)
- Rationale: Photographs "developing" from the archive

**Map Transformations**:
- Duration: 800-1200ms per transition
- Style: Borders morph, territories fade/appear
- Rationale: Territory is contested; maps are arguments

### Motion Values

```css
:root {
  /* Durations */
  --duration-fast: 200ms;      /* UI feedback */
  --duration-normal: 400ms;    /* Standard reveals */
  --duration-slow: 700ms;      /* Major transitions */
  --duration-map: 1000ms;      /* Border transformations */

  /* Easing */
  --ease-archaeological: cubic-bezier(0.4, 0, 0.2, 1);
  --ease-document: cubic-bezier(0.25, 0.46, 0.45, 0.94);
  --ease-reveal: ease-out;

  /* Parallax Speeds */
  --parallax-bg: 0.3;
  --parallax-mid: 0.6;
  --parallax-subject: 1.0;
  --parallax-overlay: 1.2;

  /* Stagger */
  --stagger-text: 50ms;
  --stagger-timeline: 80ms;
}
```

### Signature Animations

**1. The Excavation Layer Reveal**
Progress through the essay visualized as archaeological excavation—digging down through strata, each era a distinct layer.

**2. The Map Morph**
Borders transform across history: Bronze Age city-states → Israelite kingdoms → Roman provinces → Ottoman districts → Mandate borders → 1947 partition → 1949 armistice → 1967 occupation → present.

**3. The Document Materialize**
Historical documents (Balfour Declaration, UN Resolution 181, etc.) appear as if emerging from an archive—paper texture, aged edges, typewriter text.

**4. The Parallel Timeline**
1948 chapter: Two timelines advance simultaneously (Israeli independence / Palestinian Nakba). Neither dominant; both visible.

**5. The Name Overlay**
Place names layer across the landscape: Canaan, Israel, Judah, Palaestina, Palestine, Israel—translucent, overlapping, none final.

## Visual Style Guide

### Photography Approach

**Era-Specific Processing**:

| Era | Processing | Example |
|-----|------------|---------|
| Antiquity | Warm earth tones, museum documentation style | Archaeological photography |
| Classical | Cool grey, stone textures | Roman/Byzantine remains |
| Islamic | Rich, warm, respectful of architectural beauty | Dome, Al-Aqsa, tilework |
| Ottoman | Sepia, early photography grain | 1850s-1917 archives |
| Mandate | High-contrast B&W, documentary grain | 1917-1948 press photos |
| 1948 | Harsh B&W, war photography aesthetic | Conflict documentation |
| Modern | Color documentary, journalistic | Contemporary press |
| Present | Digital clarity, satellite imagery | Current documentation |

### Visual Progression

The essay moves from warm archaeological tones (antiquity) through cool classical greys, into rich Islamic colors, then sepia colonial documentation, harsh B&W conflict photography, and finally contemporary digital color. The progression is gradual—history doesn't jump cleanly between eras.

### Image Selection Principles

1. **Documentary over artistic**: Choose images that document rather than aestheticize
2. **Multi-perspective**: Include photographs from Israeli, Palestinian, and international archives
3. **People over abstractions**: Show faces, lives, daily existence—not just conflict
4. **Dignity always**: Never dehumanizing imagery of any group
5. **Attribution visible**: Every photograph sourced and credited

## Progress Bar Concept

### Concept: "The Excavation Stratigraphy"

**Visual Design**:
A vertical geological/archaeological cross-section on the left edge, showing distinct colored layers representing different eras. As the reader progresses, earlier layers are "excavated" (revealed below) while the current era remains at the surface.

**Design Details**:
- Position: Fixed left edge, vertical
- Width: 8px (expands to 12px on hover/focus)
- Layers: 12 distinct strata matching the 12 chapters
- Current position: Highlighted layer with subtle glow
- Chapter markers: Small icons representing each era

**Layer Colors** (top to bottom = present to ancient):

```
Present (Glass)        → #708090 (Slate grey)
Oslo (Plastic)         → #4682B4 (Steel blue)
Occupation (Barbed)    → #696969 (Dim grey)
1949-1967 (Rebar)      → #556B2F (Olive drab)
1948 (Concrete)        → #2F4F4F (Dark slate)
Mandate (Brass)        → #B8860B (Dark goldenrod)
Nationalism (Coal)     → #36454F (Charcoal)
Ottoman (Sandstone)    → #D2B48C (Tan)
Islamic (Tile)         → #1E4D5C (Deep teal)
Classical (Marble)     → #696969 (Grey)
Iron Age (Iron)        → #8B7355 (Bronze)
Bronze Age (Terra)     → #C65D3B (Terra cotta)
Geography (Bedrock)    → #8B4513 (Saddle brown)
```

**Justification**:
The archaeological stratigraphy metaphor is central to understanding contested history—every claim rests on layers beneath it. Digging through time is literal archaeology and metaphorical understanding.

## Layout Pattern Strategy

### Recommended Patterns

1. **Full-Bleed Photography** (`full-bleed`)
   - For: Era transitions, major historical moments, landscape/geography
   - Rationale: Documentary impact, immersive geography

2. **Split Comparison** (`split-screen`)
   - For: 1948 parallel narratives, map comparisons, before/after
   - Rationale: Essential for presenting contested perspectives simultaneously

3. **Document Focus** (`centered-document`)
   - For: Balfour Declaration, UN Resolution 181, treaty texts
   - Rationale: Historical documents as evidence

4. **Timeline Scroll** (`horizontal-timeline`)
   - For: Key dates chapters, era overviews
   - Rationale: Chronological clarity

5. **Figure Profile** (`portrait-card`)
   - For: Historical figure introductions
   - Rationale: Human-centered narrative

6. **Map Transform** (`interactive-map`)
   - For: Border changes, territorial control visualization
   - Rationale: Territory is the subject; maps are arguments

### Pattern Sequencing

- Never use same pattern consecutively
- Full-bleed for transitions between major sections
- Split-screen reserved for contested/parallel narratives
- Document focus used sparingly for key primary sources

## Mobile-Native Considerations

### Mobile-First Design

- All layouts designed for 375px width first
- Touch targets minimum 44x44px
- Content readable without pinch-zoom
- Scroll-lock sequences functional but shorter on mobile

### Typography for Mobile

- Body text: 18px minimum
- Line height: 1.7 for comfortable reading
- Paragraph length: 3-4 sentences before visual break
- Hebrew/Arabic: System fonts ensure proper rendering

### Touch Interactions

- Skip affordance: Prominent, thumb-reachable
- Progress bar: Bottom horizontal on mobile for thumb access
- Maps: Simplified for small screens; pinch-zoom disabled
- Photographs: Tap to expand with caption

### Performance Budget (Mobile)

- Initial load: <3MB
- Time to interactive: <3s on 4G
- Largest contentful paint: <2.5s
- Parallax layers: Maximum 2 on mobile
- Reduced motion: Full alternative experience

## Accessibility Requirements

### Reduced Motion

When `prefers-reduced-motion: reduce`:
- Scroll-locks become simple fade transitions
- Parallax disabled entirely
- Map morphs become crossfades
- All timing increased 50%

### Screen Reader Support

- All images have descriptive alt text including historical context
- Map transformations announced via aria-live
- Progress bar announces chapter titles
- Skip links for all scroll-lock sequences

### Color Contrast

- Text/background: Minimum 4.5:1 ratio
- All accent colors tested against dark background
- Content warnings use high-contrast red on dark

### Content Warnings

- Entry warning: Cannot be bypassed; requires explicit confirmation
- Chapter warnings: Dismissible but prominent
- Visual indicator: Red accent border, warning icon

- **Rwanda Genocide**: Single tragedy arc; this is multi-millennium scope
