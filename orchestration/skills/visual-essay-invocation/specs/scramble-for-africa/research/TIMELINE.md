# Timeline: The Scramble for Africa (1870–1914)

> **Total Events**: 35  
> **Eras**: Pre-Scramble (before 1880), Scramble Peak (1880–1900), Consolidation (1900–1914)

---

## Era 1: Pre-Scramble Context (Before 1880)

### 1869 — Suez Canal Opens
| Field | Value |
|-------|-------|
| **Date** | November 17, 1869 |
| **Location** | Egypt |
| **Significance** | Dramatically shortened route to India/Asia; Egypt became strategically vital; Britain purchased shares in 1875 |
| **Key Figures** | Isma'il Pasha (Egypt), Ferdinand de Lesseps (France) |
| **Source** | Britannica; Pakenham |

### 1870s — European Control ~10%
| Field | Value |
|-------|-------|
| **Date** | c. 1870 |
| **Location** | Coastal Africa |
| **Significance** | Europeans controlled only about 10% of Africa, mostly coastal enclaves |
| **Details** | French in Algeria/Senegal; British in Cape Colony, Gold Coast; Portuguese in Angola/Mozambique coasts |
| **Source** | Wikipedia (Scramble for Africa); Britannica |

### 1871 — Stanley Finds Livingstone
| Field | Value |
|-------|-------|
| **Date** | November 10, 1871 |
| **Location** | Ujiji, Tanzania |
| **Significance** | "Dr. Livingstone, I presume?" became famous; sparked European fascination with African interior |
| **Key Figures** | Henry Morton Stanley, David Livingstone |
| **Source** | Pakenham; Stanley's published accounts |

### 1874–1877 — Stanley Maps the Congo
| Field | Value |
|-------|-------|
| **Date** | 1874–1877 |
| **Location** | Congo River basin |
| **Significance** | Stanley traced Congo River's course; information later used by Leopold II to claim territory |
| **Key Figures** | Henry Morton Stanley |
| **Source** | Hochschild, *King Leopold's Ghost* |

### 1876 — Leopold's Geographical Conference
| Field | Value |
|-------|-------|
| **Date** | September 1876 |
| **Location** | Brussels, Belgium |
| **Significance** | Leopold II hosted conference under humanitarian guise; founded International African Association as cover for personal ambitions |
| **Key Figures** | King Leopold II |
| **Source** | Hochschild; Pakenham |

### 1879 — Anglo-Zulu War
| Field | Value |
|-------|-------|
| **Date** | January–July 1879 |
| **Location** | Zululand (South Africa) |
| **Key Events** | Battle of Isandlwana (Jan 22): Zulu victory, worst British defeat to indigenous force. Battle of Rorke's Drift (Jan 22–23): British defense. Battle of Ulundi (July 4): British victory ending war |
| **Significance** | Demonstrated African military capability; ultimately led to Zulu kingdom's destruction |
| **Key Figures** | Cetshwayo kaMpande, Lord Chelmsford |
| **Source** | Vandervort; Pakenham |

## Era 2: The Scramble Peak (1880–1900)

### 1881 — The Mahdi Rises in Sudan
| Field | Value |
|-------|-------|
| **Date** | June 1881 |
| **Location** | Sudan |
| **Significance** | Muhammad Ahmad declared himself the Mahdi; launched anti-Egyptian/British jihad |
| **Key Figures** | Muhammad Ahmad (The Mahdi) |
| **Source** | Pakenham |

### 1881 — French Protectorate in Tunisia
| Field | Value |
|-------|-------|
| **Date** | May 12, 1881 |
| **Location** | Tunisia |
| **Significance** | Treaty of Bardo established French protectorate; triggered Italian resentment (they wanted Tunisia) |
| **Key Figures** | Jules Ferry |
| **Source** | Wesseling, *Divide and Rule* |

### 1882 — British Occupation of Egypt
| Field | Value |
|-------|-------|
| **Date** | September 1882 |
| **Location** | Egypt |
| **Significance** | Britain seized control to protect Suez Canal and investments; began involvement in Sudan |
| **Key Figures** | Sir Garnet Wolseley |
| **Source** | Pakenham; Britannica |

### 1882–1898 — Samori Ture's Resistance
| Field | Value |
|-------|-------|
| **Date** | 1882–1898 |
| **Location** | West Africa (Guinea, Mali, Ivory Coast) |
| **Significance** | 16-year resistance to French conquest; longest sustained African resistance |
| **Key Figures** | Samori Ture |
| **Source** | Vandervort; UNESCO *General History* |

### 1884–1885 — Berlin Conference
| Field | Value |
|-------|-------|
| **Date** | November 15, 1884 – February 26, 1885 |
| **Location** | Berlin, Germany |
| **Significance** | 14 European nations established rules for African colonization; "effective occupation" principle; no African representation |
| **Key Outcomes** | General Act of Berlin; Congo Free State recognized as Leopold's personal property; free trade zones established |
| **Key Figures** | Otto von Bismarck (host), representatives from 14 nations |
| **Source** | General Act (primary); Wikipedia; Britannica |

### 1885 — Death of Gordon at Khartoum
| Field | Value |
|-------|-------|
| **Date** | January 26, 1885 |
| **Location** | Khartoum, Sudan |
| **Significance** | Mahdist forces captured Khartoum; General Charles Gordon killed; British humiliation, desire for revenge |
| **Key Figures** | The Mahdi, General Charles Gordon |
| **Source** | Pakenham |

### 1885 — Congo Free State Established
| Field | Value |
|-------|-------|
| **Date** | February 1885 |
| **Location** | Congo basin |
| **Significance** | Berlin Conference recognized Leopold's personal rule over Congo; not a Belgian colony but his private property |
| **Key Figures** | King Leopold II |
| **Source** | Hochschild; General Act (primary) |

### 1886 — Gold Discovered in Witwatersrand
| Field | Value |
|-------|-------|
| **Date** | 1886 |
| **Location** | Transvaal, South Africa |
| **Significance** | Largest gold deposits in history; transformed South Africa; intensified British interest in Boer territories |
| **Key Figures** | Cecil Rhodes expanded mining interests |
| **Source** | Rotberg; Number Analytics |

### 1888 — Rudd Concession
| Field | Value |
|-------|-------|
| **Date** | October 1888 |
| **Location** | Matabeleland (Zimbabwe) |
| **Significance** | Lobengula tricked into signing away mineral rights to Rhodes; basis for British South Africa Company's takeover |
| **Key Figures** | Lobengula, Cecil Rhodes, Charles Rudd |
| **Source** | Pakenham; Rotberg |

### 1889 — Treaty of Wuchale
| Field | Value |
|-------|-------|
| **Date** | May 2, 1889 |
| **Location** | Ethiopia |
| **Significance** | Signed between Menelik II and Italy; Italian version claimed protectorate over Ethiopia; Amharic version did not. Dispute led to war |
| **Key Figures** | Menelik II, Italy's representatives |
| **Source** | Marcus, *A History of Ethiopia* |

### 1890 — Brussels Anti-Slavery Conference
| Field | Value |
|-------|-------|
| **Date** | July 2, 1890 |
| **Location** | Brussels, Belgium |
| **Significance** | Leopold used conference to gain legitimacy; ironically while overseeing forced labor system |
| **Key Figures** | Leopold II |
| **Source** | Hochschild |

### 1890s — Congo Rubber Terror Peak
| Field | Value |
|-------|-------|
| **Date** | 1890s–1900s |
| **Location** | Congo Free State |
| **Significance** | Rubber quota system at peak; hands severed as proof of punishment; population halved |
| **Key Figures** | Leopold II, Force Publique |
| **Source** | Hochschild; Morel's *Red Rubber* |

### 1893 — First Matabele War
| Field | Value |
|-------|-------|
| **Date** | October–November 1893 |
| **Location** | Matabeleland (Zimbabwe) |
| **Significance** | British South Africa Company defeated Ndebele; Maxim guns proved devastating; Lobengula fled and disappeared |
| **Key Figures** | Lobengula, Cecil Rhodes, Leander Starr Jameson |
| **Source** | Pakenham; Rotberg |

### 1895 — Jameson Raid
| Field | Value |
|-------|-------|
| **Date** | December 29, 1895 – January 2, 1896 |
| **Location** | Transvaal, South Africa |
| **Significance** | Failed British attempt to trigger uprising against Boers; Rhodes implicated; worsened Anglo-Boer relations |
| **Key Figures** | Leander Starr Jameson, Cecil Rhodes |
| **Source** | Pakenham; Rotberg |

### 1896 — Battle of Adwa
| Field | Value |
|-------|-------|
| **Date** | March 1, 1896 |
| **Location** | Adwa, Ethiopia |
| **Significance** | Ethiopian forces crushed Italian invasion; 17,000 Italians killed, wounded, or captured; only African victory preserving sovereignty; Italy recognized Ethiopian independence |
| **Key Figures** | Menelik II, Empress Taytu, General Oreste Baratieri (Italy) |
| **Casualties** | Italy: 6,000 dead, 1,500 wounded, 3,000 captured. Ethiopia: 4,000–5,000 dead |
| **Source** | Marcus; Vandervort; Pakenham |

### 1896–1897 — Second Matabele War (Chimurenga)
| Field | Value |
|-------|-------|
| **Date** | March 1896 – October 1897 |
| **Location** | Rhodesia (Zimbabwe) |
| **Significance** | Ndebele and Shona uprising against British South Africa Company rule; Rhodes personally negotiated peace |
| **Key Figures** | Cecil Rhodes |
| **Source** | Pakenham |

### 1898 — Battle of Omdurman
| Field | Value |
|-------|-------|
| **Date** | September 2, 1898 |
| **Location** | Omdurman, Sudan |
| **Significance** | British/Egyptian forces destroyed Mahdist army; avenged Gordon's death; Sudan conquered |
| **Key Figures** | General Herbert Kitchener, Khalifa Abdallahi |
| **Technology** | Maxim guns killed 11,000 Mahdists in hours |
| **Source** | Pakenham |

### 1898 — Fashoda Incident
| Field | Value |
|-------|-------|
| **Date** | September–November 1898 |
| **Location** | Fashoda, Sudan |
| **Significance** | French expedition met British forces; nearly caused Anglo-French war; France backed down; led to improved relations |
| **Key Figures** | General Kitchener (Britain), Jean-Baptiste Marchand (France) |
| **Source** | Wesseling; Wikipedia |

### 1898 — Samori Ture Captured
| Field | Value |
|-------|-------|
| **Date** | September 29, 1898 |
| **Location** | Guélémou, Ivory Coast |
| **Significance** | End of 16-year resistance; Samori exiled to Gabon where he died 1900 |
| **Key Figures** | Samori Ture |
| **Source** | Vandervort |

### 1899–1902 — Second Boer War
| Field | Value |
|-------|-------|
| **Date** | October 11, 1899 – May 31, 1902 |
| **Location** | South Africa |
| **Significance** | Britain fought Boer republics; concentration camps killed 26,000 Boer civilians; British victory led to Union of South Africa |
| **Key Figures** | Paul Kruger, Lord Kitchener, Cecil Rhodes |
| **Source** | Pakenham; Britannica |

## Era 3: Consolidation and Pre-War Tensions (1900–1914)

### 1902 — Cecil Rhodes Dies
| Field | Value |
|-------|-------|
| **Date** | March 26, 1902 |
| **Location** | Cape Town, South Africa |
| **Significance** | Death of leading British imperialist at age 48; left wealth to Rhodes Scholarship |
| **Key Figures** | Cecil Rhodes |
| **Source** | Rotberg |

### 1904 — Casement Report Published
| Field | Value |
|-------|-------|
| **Date** | February 1904 |
| **Location** | London (report on Congo) |
| **Significance** | British consul's official report documented Congo atrocities; galvanized reform movement |
| **Key Figures** | Roger Casement |
| **Source** | Casement Report (primary); Hochschild |

### 1904 — Herero and Nama Genocide Begins
| Field | Value |
|-------|-------|
| **Date** | January 1904 – 1908 |
| **Location** | German Southwest Africa (Namibia) |
| **Significance** | First genocide of 20th century; 65,000 Herero (80%), 10,000 Nama (50%) killed |
| **Key Events** | October 1904: Von Trotha's extermination order; Concentration camps established |
| **Key Figures** | Lothar von Trotha, Samuel Maharero (Herero leader) |
| **Source** | Olusoga & Erichsen, *The Kaiser's Holocaust* |

### 1904 — Entente Cordiale
| Field | Value |
|-------|-------|
| **Date** | April 8, 1904 |
| **Location** | London/Paris |
| **Significance** | Settled Anglo-French colonial disputes; France recognized British Egypt, Britain recognized French Morocco |
| **Source** | Wesseling |

### 1905 — First Moroccan Crisis
| Field | Value |
|-------|-------|
| **Date** | March 1905 |
| **Location** | Tangier, Morocco |
| **Significance** | Kaiser Wilhelm II challenged French influence in Morocco; Germany demanded international conference; heightened European tensions |
| **Key Figures** | Kaiser Wilhelm II |
| **Source** | Wesseling; Wikipedia |

### 1905–1907 — Maji Maji Rebellion
| Field | Value |
|-------|-------|
| **Date** | July 1905 – August 1907 |
| **Location** | German East Africa (Tanzania) |
| **Significance** | Multi-ethnic uprising against German colonial rule; belief in maji (water) providing bullet immunity; brutally suppressed; 75,000–300,000 African deaths |
| **Key Figures** | Kinjikitile Ngwale (spiritual leader) |
| **Source** | Wikipedia (Maji Maji); UNESCO *General History* |

### 1906 — Morel Publishes "Red Rubber"
| Field | Value |
|-------|-------|
| **Date** | 1906 |
| **Location** | London |
| **Significance** | Comprehensive exposé of Congo Free State horrors; accelerated reform pressure |
| **Key Figures** | E.D. Morel |
| **Source** | Morel, *Red Rubber* (primary) |

### 1908 — Leopold Forced to Cede Congo
| Field | Value |
|-------|-------|
| **Date** | November 15, 1908 |
| **Location** | Brussels |
| **Significance** | International pressure forced Leopold to transfer Congo to Belgian state; became Belgian Congo |
| **Key Figures** | Leopold II (died 1909) |
| **Source** | Hochschild |

### 1911 — Second Moroccan Crisis (Agadir)
| Field | Value |
|-------|-------|
| **Date** | July 1911 |
| **Location** | Agadir, Morocco |
| **Significance** | Germany sent warship to Agadir; demanded compensation for French expansion; settled but increased Anglo-German tensions |
| **Key Figures** | Kaiser Wilhelm II |
| **Source** | Wesseling |

### 1912 — Morocco Becomes French Protectorate
| Field | Value |
|-------|-------|
| **Date** | March 30, 1912 |
| **Location** | Fez, Morocco |
| **Significance** | Treaty of Fez established French protectorate; Spain received smaller zone |

### 1914 — European Control ~90%
| Field | Value |
|-------|-------|
| **Date** | August 1914 |
| **Location** | All Africa |

| 1890s | Rubber terror peaks; Adwa victory; Fashoda; Boer War begins |
| 1900s | Congo reform movement; Herero genocide; Maji Maji; Morocco crises; Leopold forced out |
| 1910s | Consolidation; WWI shifts colonial control (German colonies redistributed) |
