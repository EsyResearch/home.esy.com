# Confidence Matrix: R&B — The Heartbeat That Taught Pop to Feel

> **Status**: Complete
> **Purpose**: Map claims to evidence strength for editorial decisions

---

## Confidence Levels

| Level | Definition | Source Requirement | Editorial Treatment |
|-------|------------|-------------------|---------------------|
| **HIGH** | Verified fact | 3+ independent sources, primary documentation | State as fact |
| **MEDIUM** | Well-supported | 2 reliable sources | State with attribution |
| **LOW** | Plausible | 1 source or secondary only | Qualify with "reportedly" |
| **CONTESTED** | Disputed | Sources disagree | Present multiple perspectives |

## Recommendations for Essay

### State as Unqualified Fact
- All HIGH confidence claims
- Dated events with documentation
- Organizational facts (founding dates, sales figures)

### Qualify with Attribution
- MEDIUM confidence claims
- Interpretive frameworks
- Exact wording of quotes with variation

### Present as Perspective
- Contested claims
- Analytical arguments (symbiosis theory, etc.)
- "Authenticity" analysis

### Avoid or Handle Carefully
- Exact financial figures without documentation
- Specific studio dimensions without measurement
- Claims with only one secondary source
