7. **What was the international response?** ‚Äî UN partition, armistices, UNRWA

## 3. Visual Treatment Philosophy

### Treatment Type: Photorealistic (Archival/Documentary)

| Aspect | Specification |
|--------|---------------|
| Primary Medium | Historical photography, archival maps, documents |
| Era Processing | B&W for pre-1948 and 1948 events; sepia for Mandate-era documents |
| Supplementary | Original vector diagrams, maps, timelines (NOT mixed with photos) |
| Reconstructions | Allowed ONLY when clearly labeled "Artist's reconstruction" |
| Prohibited | Graphic imagery, humiliation, sensationalism |

| Source Type | Status | Notes |
|-------------|--------|-------|

| Wikimedia Commons (CC BY / CC BY-SA / PD) | ‚úÖ Verify each file | Check license explicitly |
| USGS Landsat satellite imagery | ‚úÖ Public Domain | "Now" layer for sliders |
| Natural Earth vectors | ‚úÖ Public Domain | Base map vectors |
| UN photos | ‚ö†Ô∏è Verify | Often have restrictions |
| Getty/AP/Reuters | ‚ùå Avoid | Require paid license |
| Contemporary designer tatreez patterns | ‚ùå Avoid | Copyright protected |

### Reconstruction Policy
- AI-generated or illustrated reconstructions: **ONLY when clearly labeled**
- Label format: "Artist's reconstruction (not a photograph)"
- Reconstruction notes panel required: what is grounded vs. artistic inference
- Never present reconstructions as documentary photography
