## Licensing-Safe Source Strategy

### Tier 1 Sources (Preferred)

| Source | License | URL | Use Case |
|--------|---------|-----|----------|
| **Library of Congress Matson Collection** | Public Domain | loc.gov/collections/g-eric-and-edith-matson-photographs | 23,000+ images of Palestine 1898–1946 |
| **Palestine Open Maps** | Public Domain (UK CDPA 1998) | palopenmaps.org | Mandate-era map sheets, 1:20,000 scale |
| **Natural Earth** | Public Domain | naturalearthdata.com | Base map vectors |
| **USGS Landsat** | Public Domain | earthexplorer.usgs.gov | Modern satellite "Now" layer |
| **Wikimedia Commons (PD/CC)** | Per-file verification | commons.wikimedia.org | Historical photos, maps, documents |
| **UN Archives (verified PD)** | Per-document check | un.org/unispal | Document scans |

### Tier 2 Sources (Verify Per-File)

| Source | License | Notes |
|--------|---------|-------|
| National Library of Israel | "Any Use Permitted" per-file | Mandate-era maps; verify each file |
| British National Archives | Crown Copyright rules | May have restrictions; check |
| David Rumsey Map Collection | Varies | Historical maps; verify terms |

### Tier 3 Sources (Avoid Unless Licensed)

| Source | Why Avoid |
|--------|-----------|
| Getty Images | Paid license required |
| AP/Reuters | Paid license required |
| Contemporary designer tatreez | Copyright protected |
| UN photos without explicit terms | Often restricted |

### 8. Asset Sourcing Workflow (Gate 4.5)

1. **Search Tier 1 sources first** (LOC Matson, Palestine Open Maps, Wikimedia PD)
2. **Verify license per-file** (even within Tier 1 collections)
3. **Extract direct URLs** using image-url-extraction skill
4. **Prepare attribution text** for each asset
5. **Create original assets** where historical sources unavailable
6. **Label reconstructions clearly** ("Artist's reconstruction, not a photograph")

---

### 9. Reconstruction Policy

When historical imagery is unavailable:

**Allowed**:
- Original vector diagrams
- Map overlays and annotations
- Illustrated reconstructions IF clearly labeled

**Required Labels**:
- "Artist's reconstruction (not a photograph)"
- "Diagram based on historical accounts"
- "Map overlay showing approximate locations"

**Never Present As**:
- Documentary photography
- Primary source evidence
- Unqualified historical fact
